## Character data fields to preserve

Preserve id, name, category, themeTag, sprite, thumbnail, rarity, coinReward, unlockTier, cardNumber, particlePreset, decalPreset, impactSounds, burstSound, behaviorProfile, deformability, elasticity, and gooLevel.

## Roblox-only fields to add

Add ImageAssetId, CardImageAssetId, ModelName, BuddyModelName, Zone, FirstFeeling, SignatureSound, IsLaunchCharacter, IsStoryCharacter, and IsCoreCharacter.

## Story character flags

Flag these specially: Soft Dumpling, Goo Ball, Blushy Bun Bunny, Sparkle Mochi, Galaxy Dumpling, Glitter Goo Ball, Aurora Stretch Cube, Star-Eyed Bunny, Glow Ghost Puff, Celestial Dumpling Core, Singularity Goo Core, and Mythic Plush Familiar.
